Segundo parcial

En este parcial de fotomontaje con GIMP se creó el efecto de derretimiento, integrando una luna cálida y brillante sobre una cascada para simular que se funde con el agua. Se trabajó por capas duplicadas para conservar versiones originales, aplicando ajustes de color como Temperatura, Niveles y Curvas para lograr variaciones de luz diurna y atardecer. La luna fue importada, recortada con selección por color, y posicionada con herramientas como Mover, Escalar y Rotar. Se usó máscara de capa y pincel para ocultar bordes detrás de rocas sin dañar la imagen original. Para el efecto de resplandor se duplicó la luna, se aplicó desenfoque gaussiano y se controló con otra máscara. Para finalizar, se deformó la base de la luna con Warp Transform y se integró con la cascada usando Emborronar, Curar y pinceladas suaves. Para finalizar detalles se añadieron efectos de luz ambiental cálida y una viñeta para cerrar la composición de forma armónica y agradable.
